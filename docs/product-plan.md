## 一、用户需求分析

### 1.1 目标用户画像

**核心用户**：20-35 岁年轻女性（占比 85%）



* **职场新人**：月薪 5k-15k，有储蓄但不懂理财，害怕亏损

* **宝妈**：时间碎片化，希望利用空闲时间学习理财，为家庭积累财富

* **自由职业者**：收入不稳定，需要建立稳健的理财体系

* **高知女性**：北大等高校校友，追求品质生活，认同知识付费

### 1.2 用户核心痛点



| 痛点         | 具体表现               | 解决方案                        |
| ---------- | ------------------ | --------------------------- |
| **理财太枯燥**  | 专业术语难懂，学习过程痛苦，难以坚持 | 养猫式游戏化体验，把理财变成 "养宠物"        |
| **学习有压力**  | 怕学不会、怕考不过，有畏难情绪    | 无强制打卡、无进度条，刷视频就能学，知识树可视化进度  |
| **交易有焦虑**  | 怕亏钱、怕踩坑，不敢开始真实交易   | 零压力模拟交易，亏损无惩罚，小猫全程陪伴安慰      |
| **缺乏情感陪伴** | 理财是孤独的事，没人交流、没人鼓励  | 专属小猫陪伴，同频姐妹社区，温暖治愈的氛围       |
| **怕被割韭菜**  | 市面上理财课鱼龙混杂，容易被骗    | 平台只做教育，不荐股、不诱导投资，资金 100% 安全 |

### 1.3 用户需求优先级

✅ **P0（必须实现）**：基础养猫功能、短视频学习、模拟交易、成长金系统

✅ **P1（重要）**：知识树体系、喵系学位成长、成就系统、社区基础功能

✅ **P2（次要）**：小猫旅行、高级会员、更多小猫品种

✅ **P3（未来）**：多人猫舍、小猫繁殖、直播功能

## 二、产品整体设计

### 2.1 产品定位与核心价值

**一句话定位**：国内首个养猫式理财成长平台，让女生无痛学理财

**核心理念**：理财是载体，养猫是核心，情绪价值是目的

**slogan**：种一只小猫，赚一笔小钱

**核心价值三角**：



1. **情绪价值**：软萌小猫陪伴，治愈理财焦虑

2. **教育价值**：系统化理财知识，无痛学习成长

3. **实用价值**：零风险模拟交易，零钱通收益增值

### 2.2 核心玩法闭环



```
获得猫罐头 → 种下小猫种子 → 收获专属小猫 → 小猫陪伴+帮你搞钱 → 获得更多猫罐头
```

**猫罐头获取渠道**：



1. 刷理财短视频（1 个 / 视频）

2. 模拟交易盈利（收益的 10%）

3. 存入成长金（1 元收益 = 10 个猫罐头）

4. 社区优质内容（5-50 个 / 帖）

5. 完成成就任务（10-100 个 / 成就）

## 三、详细功能设计（PRD）

### 3.1 我的猫舍模块（P0）

#### 3.1.1 启动页与首页交互

**启动流程**：



1. 用户打开小程序 → 显示毛玻璃模糊效果

2. 隐约看到小猫在背后做动作（竖中指 / 四仰八叉睡觉 / 舔毛）

3. 屏幕中央显示「推开窗户」按钮

4. 点击后毛玻璃消失，小猫跑过来蹭屏幕，显示「应聘铲屎官」按钮

5. 点击按钮 → 小猫跳进怀里动画 + 猫叫音效 + 金币掉落 → 进入首页

**核心操作**：



* **撸猫**：点击小猫任意位置 → 随机触发撒娇动作（打滚 / 蹭手 / 呼噜声）+ 亲密度 + 1

* **喂罐头**：点击底部「喂罐头」按钮 → 小猫吃罐头动画 → 状态更新（看起来不错 / 还行 / 需要你）

* **收获小猫**：小猫种子成熟后 → 点击「收获」按钮 → 获得新小猫

#### 3.1.2 小猫性格系统（首发 4 种）



| 品种  | MBTI       | 专属技能           | 互动特点        | 解锁条件      |
| --- | ---------- | -------------- | ----------- | --------- |
| 橘猫  | ENFJ 暖心大姐姐 | 每日天气提醒、自动打理猫舍  | 话多、爱撒娇、会安慰人 | 新手默认      |
| 布偶猫 | INFP 文艺少女  | 每日写诗、旅行带回最美明信片 | 温柔、安静、多愁善感  | 完成 10 门课程 |
| 蓝猫  | INTJ 理性学霸  | 交易数据分析、客观投资建议  | 高冷、话少、逻辑清晰  | 完成 20 门课程 |
| 美短  | ESFP 元气甜妹  | 讲笑话、理财小游戏      | 活泼、好动、爱热闹   | 获得喵喵生学位   |

#### 3.1.3 小猫旅行系统（P2）



* **触发机制**：每日 30% 概率自动出发；或消耗 50 猫罐头准备行李指定目的地

* **旅行时长**：1-3 天

* **目的地**：北大燕园、《种地吧》麦田、上海外滩、奥马哈、霍尔木兹海峡

* **带回物品**：高清照片、手写明信片（含理财知识）、稀有猫罐头、限定皮肤

* **特殊设计**：给小猫买摄像机 / 无人机，可带回更高清的照片和视频

### 3.2 燕园喵庠模块（P0）

#### 3.2.1 抖音式短视频流



* **交互**：上下滑动切换，自动播放下一个，15-60 秒 / 视频

* **奖励**：完整看完 1 个视频 → 自动获得 1 个猫罐头 + 小猫语音鼓励

* **内容分类**：


  * 观鱼堂（入门）：理财基础、储蓄技巧、货币基金

  * 衔蝉书院（进阶）：基金入门、指数投资、资产配置

  * 锦毛学馆（专业）：股票基础、基本面分析、行业研究

  * 弄墨斋（大师）：家庭理财、税务规划、长期投资

#### 3.2.2 知识树体系（P1）

**核心设计**：**一棵知识树 = 一个喵喵生学位**



* **树干**：对应学科大类（如经济学基础、金融学基础）

* **树枝**：对应课程（如《货币银行学》《证券投资学》）

* **树叶**：对应知识点（如 "什么是复利"" 什么是 ETF"）

* **果实**：对应课程考试通过

**遗忘曲线机制**：



* 学习后 7 天未复习 → 对应树叶颜色变淡

* 14 天未复习 → 树叶变灰

* 重新学习该知识点 → 树叶恢复绿色

* 直观展示 "学习就是种树，遗忘就是落叶"

#### 3.2.3 喵系学位成长体系（P1）

**设计理念**：致敬高等教育体系，采用梯度化难度设计，目标用户获取比例为：**喵喵生 60%、喵士生 30%、喵士后 10%、喵师傅 1%（顶级荣誉）**，充分体现学位的稀缺性和尊重感。



| 喵系学位 | 学分要求   | 课程数量 | 解锁条件                                                        | 成长金基础上限  | 猫罐头兑换比例 |
| ---- | ------ | ---- | ----------------------------------------------------------- | -------- | ------- |
| 喵喵生  | 140 学分 | 35 门 | 1. 完成所有喵喵生阶段课程2. 每门课程刷满 80% 视频时长，系统自动推送考试机会3. 通过对应课程考试      | 10000 元  | 1:10    |
| 喵士生  | 30 学分  | 6 门  | 1. 获得喵喵生学位2. 社区获赞≥500 + 发表≥10 篇优质帖3. 刷满喵士生阶段课程 80% 时长并通过考试  | 30000 元  | 1:12    |
| 喵士后  | 20 学分  | 4 门  | 1. 获得喵士生学位2. 通过 1 门国家级金融从业资格证（以证代考）3. 刷满喵士后阶段课程 80% 时长并通过考试 | 50000 元  | 1:15    |
| 喵师傅  | 10 学分  | 2 门  | 1. 获得喵士后学位2. 连续学习≥365 天 + 社区贡献前 1%3. 完成喵师傅阶段综合考核            | 100000 元 | 1:20    |

**考试系统**：



* 考试形式：每门课程对应 10 道单选题，均为**基础名词解释和核心概念辨析**（如 "以下哪个是货币基金的特点"）

* 考试时长：10 分钟，60 分及格

* 解锁规则：**仅当用户刷满对应课程 80% 的视频时长后，系统才会弹出考试邀请弹窗**，未刷满无法参加考试

* 重考规则：考试不及格可无限次重考，每次题目随机抽取

* **以证代考**：上传真实的证券 / 基金 / 银行从业资格证、初级会计证等国家级金融证书，可直接免考对应等级的喵系学位

#### 3.2.4 成长金上限动态调整（P0）

**类似芝麻信用分的多维度评估模型，每周一更新**



| 评估维度 | 权重  | 加分项                | 减分项       |
| ---- | --- | ------------------ | --------- |
| 学习行为 | 35% | 每日学习、完成课程、通过喵系学位考试 | 连续 7 天未学习 |
| 理财实践 | 30% | 模拟交易盈利、风险控制、资产配置合理 | 频繁交易、追涨杀跌 |
| 社区贡献 | 20% | 优质内容、获赞、帮助他人       | 违规发帖、被举报  |
| 信用记录 | 15% | 连续登录、信息完整          | 违规行为      |

**调整规则**：



* 每周评分上涨 → 成长金上限 + 500-2000 元

* 每周评分下降 → 成长金上限不变（只涨不跌，保护用户体验）

* 获得喵系学位 → 直接解锁对应基础上限

### 3.3 实验田模块（P0）

#### 3.3.1 模拟交易系统



* **初始资金**：20 个猫罐头（小猫的私房钱，引导用户存入成长金）

* **交易标的**：


  * MVP 阶段：仅开放行业 ETF（美妆、煤炭、石油、电力、消费、科技）

  * 后续阶段：逐步开放港美股个股

* **交易规则**：T+0、千分之一手续费、复刻真实涨跌停

* **界面设计**：K 线图改为粉色系，隐藏复杂指标，只显示价格和涨跌幅

#### 3.3.2 小猫代打功能



* 用户可选择 "让小猫帮我交易"

* 不同性格小猫对应不同策略：


  * 橘猫：保守型，只买货币基金和宽基 ETF

  * 蓝猫：稳健型，指数基金 + 行业 ETF 轮动

  * 布偶：平衡型，股债 6:4 配置

  * 美短：激进型，行业主题 ETF 轮动

* 盈亏全部归用户，小猫不收取任何费用

#### 3.3.3 情绪价值反馈



* **盈利**：小猫跳起来撒花 → "主人太棒了！赚了好多罐头～" + 额外奖励盈利的 10% 猫罐头

* **亏损**：小猫蹭手安慰 → "没关系呀，好公司打折了，我们慢慢买～" + 无任何惩罚

* **频繁交易**：小猫翻白眼 → "交易太频繁啦，我们要做长期主义哦～"

* **大额盈利**：小猫竖中指 → "乱买的叭？运气不错～"

### 3.4 成长金模块（P0）

#### 3.4.1 核心流程



1. 点击「开通成长金」→ 直接跳转微信官方零钱通页面

2. 完成微信实名认证和绑卡（平台不触碰任何用户信息）

3. 存入任意金额 → 每日收益自动兑换为猫罐头

4. 随时可取本金，T+1 日到账

#### 3.4.2 合规设计（必须严格执行）



* ✅ 所有资金操作 100% 在微信官方页面完成

* ✅ 平台永远不显示用户真实资金余额和收益金额，只显示猫罐头数量

* ✅ 所有金融页面底部固定免责声明："本平台仅提供理财教育服务，不构成任何投资建议"

* ✅ 不承诺收益、不诱导投资、不接触用户资金

### 3.5 狸奴阿果拉社区（P1）

#### 3.5.1 核心话题板块



* \#我的小猫今天做了什么：晒小猫日常、互动瞬间

* \#今天收获了几只小猫：展示新解锁的小猫品种和皮肤

* \#燕园喵庠学到的小知识：分享学习心得、理财笔记

* \#小猫旅行明信片：晒小猫带回的照片和明信片

* \#搞钱女孩的日常：分享生活、储蓄计划、成长故事

#### 3.5.2 社区规则（红线）

❌ **绝对禁止**：讨论具体股票代码、晒收益率、荐股、诱导投资

❌ **严厉打击**：广告、诈骗、人身攻击、负面抱怨

✅ **鼓励分享**：晒猫、学习心得、成长故事、生活小确幸

**违规处理**：第一次警告，第二次禁言 7 天，第三次永久封号

#### 3.5.3 互动激励



* 优质内容获得「园丁徽章」+ 10-50 个猫罐头

* 每周评选「本周之星」，展示在社区首页，奖励 100 猫罐头

* 评论、点赞、收藏均可获得少量猫罐头

### 3.6 成就系统（P1）

**参考多邻国，设计 100 + 个成就徽章，覆盖所有用户行为**



| 成就类型     | 部分徽章示例                  | 解锁条件                                                    |
| -------- | ----------------------- | ------------------------------------------------------- |
| **学习成就** | 青铜学习者、白银学习者、黄金学习者、学霸、考神 | 完成 10/20/30 门课程、一次性通过考试、获得喵士后学位                         |

## 六、风险控制与合规保障

### 6.1 金融合规风险（最高优先级）

✅ **资金安全**：所有资金操作 100% 在微信官方页面完成，平台不触碰任何用户资金

✅ **内容合规**：不荐股、不诱导投资、不承诺收益，所有内容仅做教育用途

✅ **风险提示**：所有金融页面显著位置标注风险提示和免责声明

✅ **资质合规**：使用微信理财通持牌资质，不自行开展金融业务

### 6.2 内容安全风险

✅ **三级审核机制**：AI 初审 → 人工复审 → 用户举报

✅ **敏感词过滤**：建立金融敏感词库，自动过滤违规内容

✅ **实时监控**：7×24 小时监控社区内容，及时处理违规信息

### 6.3 技术风险

✅ **服务器扩容**：提前配置弹性扩容，应对突发流量

✅ **数据备份**：每日自动备份数据库，确保数据安全

✅ **降级方案**：核心功能异常时，自动降级为基础可用状态

✅ **应急预案**：制定服务器宕机、接口故障等紧急情况的处理流程
